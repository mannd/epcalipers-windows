Description
This is the Microsoft Windows version of EP Calipers.

EP Calipers provides electronic calipers for making measurements on
images of ECGs or other recordings.  Calculations such as mean heart
rate and corrected QT intervals can be performed directly in the app.


Installation
Unzip the zip file and double click the setup file.  This
version of EP Calipers does not require installation of Ghostscript
to open PDF files.
